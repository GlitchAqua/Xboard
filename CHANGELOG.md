# Changelog

## v0.1.1 (2026-03-28)

### ShadowVeil 私有代理协议集成
- 新增 `shadowveil` 协议类型，支持节点创建和配置（TLS/SNI/padding）
- 支持 ClashMeta、SingBox、通用 URI 三种订阅格式生成
- 管理后台节点表单支持 shadowveil 类型配置

### 用户名注册（去邮箱依赖）
- 注册仅需用户名+密码，邮箱改为选填
- 登录支持用户名或邮箱
- 个人中心新增邮箱绑定/解绑功能，绑定时强制邮箱验证码验证
- 前端登录、注册、个人中心页面同步更新

### 邮件队列管理
- 批量邮件限速 30 封/分钟（RateLimitedWithRedis）
- 事务性邮件（验证码、绑定邮箱、密码重置）不限速，优先发送
- 管理后台侧边栏新增「邮件队列」页面：队列统计、邮件日志、筛选搜索、自动刷新

### 其他
- 管理后台从 git submodule 改为直接跟踪文件
- XboardUsage 自定义主题源码及编译产物纳入版本控制
